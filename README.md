NumPy Library Project
Description

This repository contains Python projects and exercises using NumPy, the fundamental package for scientific computing with Python.
It includes Jupyter notebooks for learning and practicing array operations, mathematical computations, and data manipulations .

| File                     | Description                                      |
| ------------------------ | ------------------------------------------------ |
| `ist_np.ipynb`           | Introduction to NumPy and basic array operations |
| `problem_part_one.ipynb` | First set of NumPy problems with solutions       |
| `problem_part2.ipynb`    | Advanced NumPy exercises                         |
| `second_np.ipynb`        | Intermediate NumPy practice                      |
| `third_np.ipynb`         | NumPy functions and broadcasting examples        |


## Features
Creation and manipulation of NumPy arrays
Mathematical operations (sum, mean, multiplication, etc.)
Array reshaping, stacking, and splitting
Indexing, slicing, and conditional selection
Use of universal functions (ufuncs)
Handling multi-dimensional arrays
